# Carpeta de proyectos

Agrega aqui una carpeta por proyecto usando el mismo `slug` definido en `portfolio/catalog.py`.

Ejemplo:

```text
static/assets/projects/climatizacion-oficinas-las-condes/
static/assets/projects/ventilacion-retail-nunoa/
static/assets/projects/extraccion-cocina-centro/
```

Dentro de cada carpeta:

- `cover.jpg`
- `gallery-01.jpg`
- `gallery-02.jpg`
- `gallery-03.jpg`
